Dataset Description
This dataset contains digital image files that display the Great Lakes ice concentration using a color chart.
Each image provides a visual representation of the daily ice conditions across all five Great Lakes.
Image Information
Format: JPEG (.jpg)
Content: Each image shows ice concentration levels, with colors representing the percentage of ice cover.
Color Chart:
Light blue – Open water
The color from light blue to black – Increasing ice concentration
White – Land areas
File Naming Convention
Files follow the format:
yyyymmdd.jpg
Where:
yyyy = Year (4 digits)
mm = Month (2 digits)
dd = Day (2 digits)
Example:
20190114.jpg
represents the ice concentration map for January 14, 2019.
Data Source
The images were developed by the NOAA Great Lakes Environmental Research Laboratory (GLERL) based on Great Lakes ice concentration data.
